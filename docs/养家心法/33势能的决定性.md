# 势能的决定性 

势能的决定性是可以解释为很多的词势能、趋势等，先讲势能我们初中物理都学过惯性理论，下面是惯性的定义。我们把物体保持运动状态不变的属性叫做惯性，惯性代表了物体运动状态改变的难易程度。惯性的大小只和物体的质量有关，质量大的物体运动状态，相对难于改变，也就是惯性大，质量小的物体运动状态相对容易改变，也就是惯性小。基本解释，就是在不受外力的情况下，会继续保持当前势头。思考下这东西跟打板有没有异曲同工，上涨，股价在当前运动趋势下是有惯性的那被涨停板强行停下，后惯性依旧，跌停板也是一样，为什么我不建议朋友们抄底跌停板，既然能跌停，势能就足够大惯性也足够大。

这个不是今天要讲的重点，有句老话说得好，强者恒强，弱者恒弱，如果把强和弱抽象点看，都看成一种势能会容易理解。惯性定义里提到一个概念，质量可以理解为船大难掉头。质量大的个股既然是能起来的惯性同样也是很大，即使不能继续保持上涨的势能，但是也不会马上转为下跌，势能这点解释了这种个股在头部曲线还是平滑的。势能的力量是可怕的。众所周知很多车祸是刹车不及时引起的，尤其是大车更难快速刹停，其实都是惯性惹的祸。

趋势，上半部分讲了很多也去关于趋势的，但仅限于个股和局部的趋势，下面写写大的环境，问个问题，2000到2010年最好做的是什么？ 2010年至2015年最好做的是什么这个？问题回答太简单了，前10年绝大多数人会讲房地产，无论是投资还是个人住房到现在都收获了不错的回报，包括12,13年具体我记不太清的公众号热潮，随便一个写手的工号。做到几百万。然而现在的公众号早已满目疮痍，这就是趋势的力量，也是见识的力量引用雷军先生的一句话，站在风口上猪都能飞起来，很多人讲难以抓到所谓的势头，等着真正抓到的时候，早过去了，讲一点就足够了。过去10年的趋势是什么？
我说，互联网，没人反对吧，可能你抓不到公众号，这类大趋势里的小趋势。倘若说大趋势也看不清那还是回家种地吧，你想让资金快速膨胀，难道去做互联网以外的行业吗？不是认为别的行业不赚钱，只是在这更容易成功，如果一定要跟趋势作对，那你一定会很累，改变旧思维才是正道，学会认错当个滑头，是幸运的人生也好股市也罢，谁也不会永远是对的。百年之前最伟大的投机家，利弗莫尔就提出来市场只有一面，既不是多头的一面，也不是空头的一面，而是正确的一面，市场不会犯错，即使错误，也是需要我们来用亏损来承受的。

看错小趋势的时候学会认错，趋势发生拐头的时候学会认错，势头未起的时候学会认错，有时候死扛是错误，不一定是趋势不到，只是这一路容易失去太多东西，很多走下跌通道的个股真别死扛学会认错。这去换回股市上来说，我先讲去感悟。每个人的认知，永远是对的，不对的只是时间，你现在告诉我哪只股票能涨或者涨停我都信，前提是别说时间，说了时间，这是准没戏。因为这起是根据经验反应的出来的，人都有个毛病，就是盲目崇拜和否定自己。当时我炒股刚刚入行时候，每天就爱干一件事吗？ 骂庄家，骂主力，现在这去看来真是可笑，但争取的单子确实应该长可结果已经摆在这个地方懂得适应市场学会在合适的时候认错。认错，真不丢人，亏钱才丢人。
